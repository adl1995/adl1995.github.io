Building RPM packages with rpmbuild, Koji, and GitLab-CI

Mon 06 May 2019

The RPM system facilitates in querying and updating a software package, examining package interdependencies, verifying package file permissions, and can target multiple architectures. This blog post will describe the process of building an RPM using the rpmbuild tool and will then explain how to schedule build tasks using koji. The RPM build pipeline will then be automated using continuous integration in GitLab.
